## 🚕 Navigo 
A blockchain-based cab booking website, revolutionizing transportation through secure, transparent, and decentralized transactions. Implemented smart contracts for automated payments, ensuring transparency in fare calculations.

## 📦 Technologies
- `Next.js`
- `Tailwind CSS`
-  `Sanity.io`
-  `Mapbox.gl`

## ✨ Features

- **Metamask Integration**: Seamlessly integrates Metamask for decentralized payment processing, ensuring secure and transparent transactions for users.

- **Sanity.io Database**: Leverages Sanity.io for robust and flexible database management, facilitating efficient storage and retrieval of user and trip data.

- **TailwindCSS Styling**: Utilizes TailwindCSS for sleek and responsive styling, ensuring a seamless user experience across devices and screen sizes.

- **Mapbox.gl Integration**: Integrates Mapbox.gl for real-time map functionality, providing users with accurate and up-to-date location information during their journey.

- **Decentralized Login with Metamask**: Enables users to log in using Metamask, offering a decentralized authentication method that enhances security and privacy.

- **Dynamic Fare Calculation**: Calculates fares in Ethereum based on pick-up and drop-off locations, providing users with transparent and real-time pricing information.

- **User-friendly Interface**: Offers an intuitive interface for users to enter pick-up and drop-off locations, simplifying the booking process.

- **Real-time Ethereum Amount Display**: Displays the Ethereum-based fare amount to users in real-time, ensuring transparency and clarity in pricing.

- **Responsive Design**: Implements responsive design principles with TailwindCSS, guaranteeing a seamless experience across various devices and screen sizes.

- **Secure and Transparent Transactions**: Ensures secure and transparent transactions through Metamask integration, leveraging blockchain technology for payment processing.

- **Efficient Trip Management**: Manages trips efficiently with Sanity.io database, enabling quick access to trip data for both users and drivers.

## 🚦 Running the Project
First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.
